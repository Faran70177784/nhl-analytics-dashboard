# NHL Analytics Dashboard

## Project Overview

This project is an interactive NHL Analytics Dashboard developed using Python and Streamlit. The dashboard provides insights into NHL player and team performance using data visualization and interactive filters.

## Features

- Interactive dashboard
- KPI cards
- Dynamic filters
- Correlation heatmap
- Scatter plots
- Team analysis
- Position analysis
- Download filtered data option

## Technologies Used

- Python
- Pandas
- Streamlit
- Matplotlib
- Seaborn

## Dataset

Dataset Used: NHL Game Data (skaters.csv)

## How to Run

Install required libraries:

pip install -r requirements.txt

Run the dashboard:

python -m streamlit run app.py
